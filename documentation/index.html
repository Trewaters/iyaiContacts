Getting Started

Coding iyai contact CMS. Input contact information with a web app. This is a practice project. My end goal is to use origami for the iyai CMS but I ran into some knowledge issues (things I dind't know how to do), so I decided to make an angular site while I waited. This is it.

What the app should do?

  • Start with npm start to use mongoDB. must run an npm run build to update files.

Allows you to create contacts, display all contacts, search for one contact, and import contacts from Zoho.

Development server

Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.
